# GBA Emulator (WIP)  This is a hobby Game Boy Advance emulator written in C#.  Current Features: - Basic ARM CPU structure - Memory mapping (RAM, VRAM, ROM) - Basic graphics mode rendering - Emulator loop  Planned: - More CPU instructions - Input handling - DMA - Interrupts  Purpose: Learning low-level systems programming and emulator development.
